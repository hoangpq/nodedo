## An application based on [Cyclejs](https://cycle.js.org/)
 
## Installation
```javascript
yarn --no-lockfile
```

## Usage
```javascript
yarn start
```

## Dependencies

[Cyclejs](https://cycle.js.org/)

[Webpack](https://webpack.js.org/)

[Express](https://expressjs.com/)

[Sass](http://sass-lang.com/)
